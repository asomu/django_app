Control Desk

Test Control

  • Auto-Scroll (F11) : 테스트 결과 자동 스크롤
  • Pause Test (F12) : 테스트 일지 정지 (Running 중에만 활성화)
  • Pause for 10 seconds (Ctrl + F12) : 10초간 테스트 일시정지 (Running 중에만 활성화)

UDR

  • Make UDR Sheet : UDR Directory로부터 Rx Common Plan에 사용하는 UDR Sheet를 생성한다.
  • Change Format : R&D UDR(*.csv)과 NPI UDR(*.txt)간의 상호변환

Oscilloscope

  • Capture Screen : OSC의 화면을 Capture 후 저장한다.

Loss File

  • Make Loss File From S2P : S2P File들을 선택하여 Loss File로 변환한다.

Manual Command & Response

  • 직접 명령어로 장비를 제어할 수 있다.

Spectrum Analyzer

  • Capture Screen : SA의 화면을 Capture 후 저장한다.
  • Get Trace Data : SA의 Trace를 CSV 파일로 저장한다.

Mini-Circuits SP6Tx2

  • SP6T 두 개가 붙은 파란색 Switch Box를 컨트롤한다.

Network Analyzer

  • Download SNP File : NA로부터 SNP 파일을 얻어온다.
  • Cont ON : SNP 파일을 다운로드 받은 후 NA를 Continuous로 할 것인지 결정한다.
  • Download State : NA의 상태를 파일로 저장한다.
  • Upload State : NA의 상태 파일을 업로드한다.

Noise Figure Analyzer

  • Download State : NFA의 상태를 파일로 저장한다.
  • Upload State : NFA의 상태 파일을 업로드한다.

Signal Generator

  • Get All Waveforms : SG에 있는 모든 Waveform을 리스팅한다.
  • Set Copied Waveform : 현재 클립보드에 복사된 Waveform을 SG에 설정한다.

Maury Tuner

  • Initialize Tuner : Tuner를 초기화한다.
  • Set Freq : Tuner 주파수 설정
  • Set VSWR/Phase : Tuner의 VSWR/Phase를 이동한다.
  • Combine Tuner Cal File (*.tun) : Tune Calibration File들을 합친다.

Power Supply

  • Calibrate Current : Power Supply의 Leakage Current을 측정하여 Current를 읽을 때 이 값만큼 제외하고 읽게 한다.
  • Calibration 후 정확도가 상당히 상승하는 것을 확인할 수 있다.
Calibration 시 Power Supply에는 아무것도 연결하지 않아야 한다
