Start Matrix

설치 방법

  1. 모델 압축파일을 받아서 적당한 곳에 압축을 푼다.
    • 맨 처음 압축을 풀면 Model.xml 파일 하나만 존재한다.
  2. Matrix 압축파일을 받아서 적당한 곳에 압축을 푼다.
  3. Matrix.exe를 실행하고 먼저 압축을 풀었던 모델 폴더를 선택한다.
    • 맨 처음 압축을 풀면 Model.xml 파일 하나만 존재한다.

메인 화면

  • 테스트 설정(F8) : 테스트 설정창을 보여준다. (Test Flow 참조)
  • MIPI 창 열기(F10) : 현재 선택된 모델의 MIPI GUI를 보여준다.
  • Control Desk 열기(F7) : Control Desk 창을 보여준다. (Control Desk 참조)
  • 테스트 코멘트창 : 여기에 쓴 코멘트 이름으로 서브디렉토리가 생성되어 그 안에 데이터가 저장된다.

모델 변경

  • Ctrl + '0'를 누르거나 타이틀 바를 우클릭하여 Select Model을 선택하면 다른 모델 폴더로 이동할 수 있다.

단축키

  • F1 : 프로그램 정보
  • F9 : 테스트 시작
  • ESC : 테스트 즉시 멈춤
  • F8 : 테스트 Flow 설정창 열기
  • F10 : 모델 MIPI GUI 열기
  • F7 : Control Desk 열기
  • F11 : 자동스크롤 토글
  • F12 : 테스트 일시정지 토글
  • Ctrl + F12 : 10초간 테스트 일시정지
  • Ctrl + F5 : SG에 있는 모든 Waveform을 리스팅한다. (Control Desk의 Get All Waveforms)
  • Ctrl + F6 : 현재 클립보드에 복사된 Waveform을 SG에 설정한다. (Control Desk의 Set Copied Waveform)
  • Ctrl + '0' : 모델 변경
  • Ctrl + 'A' : 테스트 결과 전체선택
  • Ctrl + ' ` ' ('1'번 키 옆) : 맨 윗줄(헤더)을 제외한 테스트 결과 전체선택
  • Ctrl + 'C' : 선택된 테스트 결과 복사
  • Ctrl + '1'~'9' : 선택된 테스트 결과 중 N번째 열(세로 Column)만 복사, 특정 데이터만 복사할 때 유리
