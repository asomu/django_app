Test Flow Management

용어 설명

  • 테스트 아이템 : 독립적으로 특정한 역할을 하거나 특정한 테스트를 진행하는 개체
  • 테스트 Flow : 테스트 아이템들을 여러개 모아서 만든 집합으로 파일 형태로 관리됨

Flow 설정 화면

  • Flow 선택창 : 사용하고자 하는 Flow를 Load하는 콤보박스이다.
  • Flow 관리 버튼들 : Flow를 복사, 삭제, 병합할 때 사용한다.
  • 테스트 장비 설정 : Flow당 하나만 존재하는 특수 아이템을 보여주며 이 아이템은 현재 Flow의 장비 및 옵션을 설정하는 역할을 한다. (Instrument 시트 참조)
  • 아이템 리스트창 : 현재 Flow를 구성하고 있는 아이템들을 보여준다.
  • 아이템 세부설정창 : 현재 선택된 아이템의 각종 옵션들을 보여주고 수정할 수 있다.

아이템 추가

  • Edit 모드에서 Add 버튼을 누르면 아이템 선택창이 뜬다.
